# Oil-and-Gas-Exploration-GHG-Calculator
Three case studies of historical oil and gas exploration projects, each with a calculator to estimate GHG output for the life of the project.
I would like to know if you use the tool or improve upon it. 
The idea for a GHG calculator for oil and gas exploration activities came up in 2025 at San Francisco Climate Week, at 9Zero. I spoke with some consultants, learned a little about intervention accounting, and started saving my pennies to hire a developer to help me build it. On 2026-06-25, I vibe coded the tool I was thinking about using Claude. This is the result, three divergent case studies and the ability to modify the calculator inputs.
I want to continue to improve the tool, but I don't know very much about GHG or public databases that could inform the output better.
